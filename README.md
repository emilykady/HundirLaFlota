# Hundir la Flota.

Bienvenido al juego de estrategia naval.
Este es un proyecto desarrollado en **Python** que te permite enfrentarte a la máquina.

## Características
* **Modo de juego:** Jugador vs. máquina.
* **Generación aleatoria:** La máquina coloca sus barcos de forma estratégica y aleatoria.
* **Sistema de coordenadas:** Interfaz intuitiva basada en ejes X e Y.
* **Validación de disparos:** Control de errores para evitar disparos repetidos o fuera del tablero.
